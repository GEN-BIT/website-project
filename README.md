# 🌐 Website Project

A modern, responsive multi-page website built using HTML, CSS, and JavaScript.

## 🚀 Live Demo

👉 https://69d6932c4a68e2c16b584076--acme-bryton-maston.netlify.app

## 📌 Features

* Responsive design (mobile + desktop)
* Multiple pages (Home, About, Services, Gallery, Contact)
* Clean UI/UX layout
* Custom styling with CSS
* Interactive elements with JavaScript

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

* `/css` → Stylesheets
* `/js` → JavaScript functionality
* `/assets` → Images and media

## ⚡ Getting Started

Clone the project:

```bash
git clone https://github.com/GEN-BIT/website-project.git
cd website-project
```

Open `index.html` in your browser.
